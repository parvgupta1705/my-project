JavaScript Explanation – Dynamic Product Filter

1. Selecting Elements

const filter = document.getElementById('categoryFilter');
→ Selects the dropdown element (<select>) so we can track when the user changes the category.

const products = document.querySelectorAll('.product');
→ Selects all product items (divs with the class .product) to show or hide them later.

2. Event Listener

filter.addEventListener('change', () => { ... });
→ Adds a listener that waits for the dropdown value to change.
→ Every time the user selects a new category (Clothing, Electronics, Books, or All), the code inside will run.

3. Getting Selected Value

const selected = filter.value;
→ Stores the chosen category (like "clothing", "electronics", "books", or "all").

4. Filtering Logic

products.forEach(product => { ... });
→ Loops through every product in the list.

Inside the loop:

if (selected === 'all' || product.dataset.category === selected)
→ Checks whether the chosen category matches the product’s data-category (like "clothing") OR if "all" is selected.

product.classList.remove('hidden');
→ Shows the product by removing the CSS class .hidden.

else { product.classList.add('hidden'); }
→ Hides products that do not match by adding the .hidden class (which has display: none;).
